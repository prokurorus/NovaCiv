# NovaCiv — визуальный стиль для медиа

## Базовая идея
Светлый минимализм, ощущение будущего без агрессии.
Фон — почти чисто белый, всё объёмное — мягко серое, как барельеф.

## Цвет
- Фон: #FFFFFF
- Тени и рельеф: мягкие серые оттенки (примерно #D4D4D4 / #E5E5E5)
- Текст: чёрный или тёмно-серый (#111111 / #27272A)

Без насыщенных цветов, без кричащих элементов.

## Формат роликов
- Shorts: 1080x1920, вертикально.
- Основной объект (андроид, лицо, силуэт) — слева или справа.
- Достаточно пустого места для текста.

## Текст в роликах
- Короткие смысловые фразы.
- Максимум 3–5 строк на экран.
- Без лозунгов и рекламы, только суть.

## Подпись
В конце ролика:
- Фраза: «Подпишись на будущее планеты» (или локальный перевод).
- Адрес сайта: novaciv.space

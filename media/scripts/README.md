# Media Scripts (пока только план)

В этой папке будут храниться скрипты для автоматизации:

- сборки роликов из пресетов (`media/shorts-presets`)
- использования голосовых профилей (`media/voices/openai`)
- конвертации текстов (`media/subtitles`) в субтитры или аудиодорожки
- планировщика публикаций (TikTok, YouTube Shorts и т.п.)

Пока здесь только структура.
Вся логика будет добавляться по мере готовности.
